# 비동기 메서드 테스트하기
앞서 간단한 메서드들에 대한 유닛 테스트를 작성해보았습니다! 이제 유닛 테스트에 대해서 어느 정도 감이 잡히셨을 것 같습니다.    
그러나, 매번 이렇게 테스트를 작성하기가 수월한 코드들이면 좋으련만... 실제 프로젝트에서는 테스트를 하기에는 이보다 복잡한 코드가 등장해서 어렵게 느껴지기도 합니다.

이번 챕터와 다음 챕터에서는 조금 새로운 테스트를 접해보도록 하겠습니다! 여러 유형으로 나눌 수 있겠지만 대표적으로 `비동기 메서드`와 `네트워킹 메서드`를 테스트하는 방법에 대해서 알아보겠습니다 :)
